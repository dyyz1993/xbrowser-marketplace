## 2. CLI жіЁеҶҢдёҺи®ӨиҜҒжөҒзЁӢ

### 2.1 еҪ“еүҚе®һзҺ°зҠ¶жҖҒ

| еҠҹиғҪ | зҠ¶жҖҒ | е®һзҺ°дҪҚзҪ® |
|------|------|----------|
| Web жіЁеҶҢ | вң… е·Іе®һзҺ° | `POST /api/auth/register` |
| Web зҷ»еҪ• | вң… е·Іе®һзҺ° | `POST /api/auth/login` вҶ’ иҝ”еӣһ API Key |
| Token йӘҢиҜҒ | вң… е·Іе®һзҺ° | `GET /api/auth/verify` (Bearer token) |
| CLI register | вқҢ жңӘе®һзҺ° | йңҖж–°еўһ CLI е‘Ҫд»Ө |
| CLI login | вқҢ жңӘе®һзҺ° | йңҖж–°еўһ CLI е‘Ҫд»Ө |
| CLI whoami | вқҢ жңӘе®һзҺ° | йңҖж–°еўһ CLI е‘Ҫд»Ө |
| Token еӯҳеӮЁ | вқҢ жңӘе®һзҺ° | йңҖе®һзҺ° `~/.xbrowser/auth.json` |

### 2.2 ж–№жЎҲиҜ„дј°пјҡCLI жіЁеҶҢ

#### Option A: CLI дәӨдә’ејҸжіЁеҶҢ

```bash
$ xbrowser plugin register
? Username: myname
? Email: my@email.com
? Password: ********
? Confirm password: ********
вң“ Registered successfully!
вң“ Your API token: xbr_abc123def456...
вң“ Token saved to ~/.xbrowser/auth.json

You can now publish plugins with: xbrowser plugin publish
```

**дјҳзӮ№**: е…ЁзЁӢ CLI е®ҢжҲҗпјҢејҖеҸ‘иҖ…дҪ“йӘҢеҘҪ
**зјәзӮ№**: CLI дёӯеҜҶз Ғиҫ“е…ҘдҪ“йӘҢдёҚеҰӮ Webпјӣж— жі•еҒҡ CAPTCHAпјӣйӮ®з®ұйӘҢиҜҒжөҒзЁӢеӨҚжқӮ

#### Option B: Web жіЁеҶҢ + CLI зҷ»еҪ•

```bash
# жӯҘйӘӨ 1: еңЁжөҸи§ҲеҷЁдёӯжіЁеҶҢ
# и®ҝй—® https://xbrowser.dev/register
# еЎ«еҶҷиЎЁеҚ•пјҢйӘҢиҜҒйӮ®з®ұ

# жӯҘйӘӨ 2: иҺ·еҸ– API Key
# зҷ»еҪ•еҗҺиҝӣе…ҘејҖеҸ‘иҖ…и®ҫзҪ®йЎөйқўпјҢеӨҚеҲ¶ API Key

# жӯҘйӘӨ 3: CLI зҷ»еҪ•
$ xbrowser plugin login --token xbr_abc123def456...
вң“ Logged in as myname
вң“ Token saved to ~/.xbrowser/auth.json
```

**дјҳзӮ№**: Web жіЁеҶҢеҸҜд»ҘеҒҡжӣҙдё°еҜҢзҡ„йӘҢиҜҒпјҲйӮ®з®ұйӘҢиҜҒгҖҒCAPTCHAпјү
**зјәзӮ№**: йңҖиҰҒеҲҮжҚўжөҸи§ҲеҷЁпјҢжӯҘйӘӨиҫғеӨҡ

```bash
# ж–№ејҸ A: CLI зӣҙжҺҘжіЁеҶҢпјҲйҖӮеҗҲеҝ«йҖҹдҪ“йӘҢпјү

### 2.3 и®ӨиҜҒжһ¶жһ„

#### еҪ“еүҚжһ¶жһ„

```
ејҖеҸ‘иҖ…иЎЁ (developers)
в”ңв”Җв”Җ id: UUID
в”ңв”Җв”Җ username: unique
в”ңв”Җв”Җ email: unique
в”ңв”Җв”Җ passwordHash: bcrypt
в”ңв”Җв”Җ role: 'developer' | 'super_admin'
в””в”Җв”Җ apiKey: UUID (дҪңдёә Bearer token дҪҝз”Ё)
```

- зҷ»еҪ•ж—¶иҝ”еӣһ `apiKey` дҪңдёә token
- и®ӨиҜҒдёӯй—ҙд»¶йҖҡиҝҮ `apiKey` жҹҘиҜў `developers` иЎЁйӘҢиҜҒиә«д»Ҫ
- Token ж јејҸ: `xbr_` еүҚзјҖ + UUID (еҪ“еүҚе®һзҺ°дёәзәҜ UUIDпјҢе»әи®®ж·»еҠ еүҚзјҖ)

#### и®ӨиҜҒж–№ејҸеҜ№жҜ”

| з»ҙеәҰ | API Key (еҪ“еүҚж–№жЎҲ) | JWT |
|------|-------------------|-----|
| е®һзҺ°еӨҚжқӮеәҰ | дҪҺ вң… | дёӯпјҲйңҖиҰҒ refresh жңәеҲ¶пјү |
| ж— зҠ¶жҖҒ | еҗҰпјҲйңҖжҹҘеә“пјү | жҳҜ |
| иҝҮжңҹз®ЎзҗҶ | ж— иҝҮжңҹ вң… з®ҖеҚ• | йңҖиҰҒеҲ·ж–°жңәеҲ¶ |
| ж’Өй”Җ | зӣҙжҺҘеҲ йҷӨ key вң… | йңҖиҰҒ blacklist |
| еӨҡи®ҫеӨҮ | еӨ©з„¶ж”ҜжҢҒ | йңҖиҰҒз®ЎзҗҶ |
| е®үе…ЁжҖ§ | дҫқиө– HTTPS | дҫқиө– HTTPS + еҜҶй’Ҙз®ЎзҗҶ |
| йҖӮеҗҲеңәжҷҜ | CLI/жңҚеҠЎй—ҙ | зҹӯжңҹ Web session |

**з»“и®ә**: еҪ“еүҚ API Key ж–№жЎҲйҖӮеҗҲ CLI еңәжҷҜпјҢдҝқжҢҒдёҚеҸҳгҖӮWeb еүҚз«ҜеҸҜиҖғиҷ‘еҸ еҠ  JWT з”ЁдәҺ session з®ЎзҗҶгҖӮ

// src/server/services/npm-registry-service.ts (ж–°еўһ)

const NPM_REGISTRY = 'https://registry.npmjs.org';

export async function searchNpmPlugins(query: string) {
  const url = `${NPM_REGISTRY}/-/v1/search?text=${encodeURIComponent(query + ' xbrowser-plugin')}`;
  const res = await fetch(url);
  const data = await res.json();

  return data.objects
    .filter((obj: any) => obj.package.keywords?.includes('xbrowser'))
    .map(formatNpmResult);
}

export async function getNpmPackageInfo(packageName: string) {
  const url = `${NPM_REGISTRY}/${encodeURIComponent(packageName)}`;
  const res = await fetch(url);
  return res.json();
}

export async function getNpmTarballUrl(packageName: string, version: string) {
  const info = await getNpmPackageInfo(packageName);
  return info.versions[version]?.dist?.tarball;
}
```
